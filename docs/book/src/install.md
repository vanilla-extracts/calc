# Introduction

## Installation
You can install `mini-calc` via several methods

### Cargo
```sh 
cargo install mini-calc
```

### Source
```sh 
git clone https://github.com/vanilla-extracts/calc
cd calc 
cargo build --release
./target/release/mini-calc
```

### Nix
```sh 
nix run github.com:vanilla-extracts/calc
```
